================
ESTop
================

Monitor ElasticSearch Tasks

Requirements
------------

* The following program works both on Linux and OSX.

Installation
------------

* Install the package into your project's virtual environment :

  ``pip install estop``

Usage
-----

* To start ESTop, just type the command :

  ``estop``
